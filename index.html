SPIRALIZER

URL
Only direct image links work. Unsplash is a great resource for pretty images.
rDiv
Rate of change in the size. The higher the number, the less they grow.
Angle
Changes the angle of the spiral. The lower the number, the tighter the spiral.
Resolution
Quantity of circles the image is divided into. Capped at number of pixels in photo.
Radius
Changes how quickly the spiral expands. The higher the number, the quicker.
Speed
Changes how many circles are created within each frame
Direction
Changes the direction in which the image is scanned
