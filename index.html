SPIRALIZER

URL
Only direct image links work. Unsplash is a great resource for pretty images.
rDiv
Rate of change in the shape size. The higher the number, the less they grow.
Angle
The difference in angle (in degrees) between each step
Resolution
Quantity of shapes the image is divided into. Capped at number of pixels in photo.
Radius
Changes how quickly the spiral expands. The higher the number, the quicker.
Speed
Changes how many shapes are created within each frame
Direction
Changes the direction in which the image is scanned
Shape
Changes the shape that is drawn each iteration
Rotation
The amount of rotation of the shape each iteration
